# [요약본] 감사 지적사례 빅데이터 기반 학교 종합감사 사전 자체점검 시스템 「감사내비」

**디지털업무개선 연구대회(충청북도교육청 도대회)** · 적용 학교: 한천초등학교(2026 종합감사 대비)
**핵심어**: 학교 자체감사, 지적사례 분석, 사전 자체점검, 데이터 기반 행정, 무설치 웹도구

---

## 1. 연구 배경 및 목적

학교는 3년 주기로 종합감사를 받지만, 준비는 담당자의 경험·기억에 의존해 **매번 처음부터 반복**된다. 인사이동 시 노하우가 단절되고, "무엇을 점검해야 하는지"부터 막막하다. 그러나 감사 지적 결과는 충청북도교육청 감사관의 『자체감사 사례집』 등으로 **이미 공개**되어 있다 — 다만 수백 쪽 PDF에 흩어져 활용되지 못할 뿐이다.

본 연구는 ① 이 공개 자료를 분석해 **빈출 지적 패턴**을 도출하고, ② 학교가 감사 전에 스스로 점검하는 **디지털 도구를 개발**하며, ③ 실제 학교에 **적용해 효과를 검증**하는 것을 목적으로 한다.

## 2. 데이터 분석 결과 — "어디서 주로 지적되는가"

충북교육청 감사관 발간 **3종 공식 자료**(2025 자체감사 사례집·분야별 착안사항·감사실무매뉴얼)의 지적사례를 코딩·분석하였다.

> **11개 분야 · 80개 점검 주제 · 점검문항 415개 · 지적사례 325건** 구조화

- **회계·재정 영역에 지적이 집중**: 공립학교 관련 237건 중 학교회계·예산(60) · 인사·복무(56) · 교무·학사(46) · 보수·수당(31) 네 분야가 대부분.
- **"돈과 근태를 다루는 절차"의 실수가 단골**: 학교회계 집행(13) · 휴가/유연근무(12) · 계약제교원 임용(10) · 예산편성(10) · 업무추진비(10) 순.
- 같은 유형이 3건 이상 반복된 **빈출(★) 주제 49개**를 우선 점검 대상으로 도출.

## 3. 개발 산출물 — 무설치·무서버 웹도구 「감사내비」

분석 결과를 탑재한, **설치·서버·DB 없이 브라우저만으로 동작**하는 도구를 개발했다. 학교 정보는 PC 밖으로 전송되지 않으며(localStorage 저장), **단일 파일(`감사내비.html`)** 로 파일 하나만 전달해 배포한다.

| 기능 | 내용 |
|------|------|
| 대시보드 | 감사 D-day, 수감자료 제출 마감 자동 계산, 준비 일정(D-day 역산), 빈출 우선 점검 추천 |
| 분야별 자체점검 | 80개 주제 415문항, 실제 지적사례·근거 법령(국가법령정보센터 링크), 역할별 보기 |
| 감사 종류별 범위 | 종합·재무·복무·특정·보안·재물조사·일상 7종 선택 시 해당 분야만 자동 조정 |
| 수감자료 보드 | 매뉴얼 기준 기본 준비물 불러오기, 부서별 담당·준비 상태 관리 |
| 통계·보고서 | 사용 효과 자동 측정, 빈도 집계표, 점검 결과·인수인계 보고서 인쇄/PDF |

## 4. 적용 및 기대 효과

한천초등학교 **2026학년도 종합감사(7/2 개시)** 대비에 적용하였다. 감사 예정일 입력만으로 수감자료 제출 마감(6/22)과 준비 일정이 자동 역산되고, 업무분장별 분담 점검과 빈출 우선 점검이 이루어졌다. 효과는 도구가 자동 기록하는 **준비도 변화·점검 시간·완료율**과 사용 교사 설문으로 측정한다.

- **정보 탐색 비용 절감**: 수백 쪽 사례집 통독 없이 "우리 학교가 점검할 항목"이 즉시 제시.
- **우선순위 판단 지원**: 빈도 데이터로 위험 높은 회계·복무 영역에 준비 역량 집중.
- **인수인계 단절 해소**: 점검 메모·인수인계 보고서로 노하우 문서화.
- **확산성**: 단일 파일로 인근 학교 전달, 데이터 교체만으로 타 시·도 확장 가능.

## 5. 결론

감사 지적사항은 이미 공개된 데이터이며, 이를 구조화하면 강력한 예방 도구가 된다. 「감사내비」는 충북 학교 감사 지적사례 325건 분석을 "우리 학교가 지금 점검할 항목"으로 변환하여, 경험에 의존하던 감사 준비를 **데이터 기반의 반복 가능한 절차**로 전환한다.
